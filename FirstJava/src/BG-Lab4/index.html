Design

There are 4 basic design principles:

  1. Contrast
  2. Repetition
  3. Alignment
  4. Proximity

The principle of contrast states that if something is different make them really different. Make them stand out so it draws attention. Can create contrast with color, fonts, sizes, whitespace etc.

The principle of repetition states that you repeat some aspect of the design with the purpose to unify and add visual interest. The repetitive element can be a font, a bullet, a line, colour, design element, spatial relationship etc. Repetition creates consistency.

The alignment principle states that nothing should be placed on the page arbitrarily. The goal is to unify and organize. Every item should have a visual connection with something else on the page.

The principle of proximity states that you group related elements together. Move items close to each other so they are seen as one unit. It implies a relationship and creates a path for eye to follow.

Accessibility

  • Covers not just deaf and blind but limited mobility, slow dialup, different devices
  • Use alt attributes on images
  • Use text that makes sense when read out of context
  • Use semantic markup
  • separate content from presentation to reduce amount of code in html file
  • Make sure color alone is not used to convey meaning
  • Ensure line by line reading of any tables makes sense

SEO

  1. Validate your site - makes it easier for search engines to read your pages, search engines read page line by line and if there are errors can prevent spider from reading the content. Valid html tends to lessen the amount of code on the page
  2. Use alt attributes on images so search engine understands what image is about. Create unique relevant description of image being described. Never repeat same alt tags unless its same image. Use keywords in your descriptions.
  3. Conduct useful and thorough Keyword Research and Optimization utlizing things like relevant keywords, and phrases to target the given audience. Incorporate these words naturally, using titles, body text, alt image text and headings. Also avoid trying to cram too much info, and rely more on high quality content for the user.
  4. Utilize Quality Content Creation - publish high qualiy, and engaging content that adressed the needs of the audience. Regularly update your website with new content, such as blog posts, articles, videos, and infographics. Good content encourages more user engagement and reduces the bounce rates.
  5. On-page SEO: On-Page SEO allows on page elements, such as meta titles, descriptions, headers h1/h2, and URL structure to allow for uniqueness to the page.
  6. Focus on building a healthy backlink profile by acquiring high-quality, relevant links from good websites in your industry. You can do this through content partnerships, collaborations, and participation in online community.
  7. MAKE IT MOBILE FRIENDLY A mobile friendly and responsive website is prioritized more by google and other search engines due to their adaptability to platform. A good mobile and user experience also increases the satisfaction of the user.
